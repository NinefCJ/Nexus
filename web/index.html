📦 模块详解

核心模块 功能介绍

两大核心模块协同工作，提供完整的命令辅助能力

📱

Android 模块

Nexus-Android

  • MainActivity - 应用入口和主界面管理
  • MainViewModel - 状态管理和业务逻辑
  • FloatingWindowService - 悬浮窗服务
  • CommandHelper - JNI 调用封装
  • ID 翻译库 - 方块/物品/音效/粒子/动画
  • 主题系统 - 8种预设主题 + 自定义
⚡

C++ 核心模块

Nexus-Core

  • Tokenizer - 词法分析器，分解命令 Token
  • Parser - 语法解析器，构建语法树
  • Completion - 智能补全引擎
  • Command Registry - 命令注册表
  • Highlighter - 语法高亮引擎
  • RapidJSON - 高效 JSON 解析

❓ 常见问题

你可能想 了解的问题

遇到问题？这里可能有你想要的答案

Nexus 支持哪些 Minecraft 版本？ ▼

Nexus 专为 Minecraft 基岩版 (Bedrock Edition) 设计，支持 1.18+ 版本。Java 版的命令语法有所不同，暂不支持。

如何在游戏中使用悬浮窗？ ▼

在设置中开启悬浮窗权限后，点击悬浮窗按钮即可显示。悬浮窗支持最小化/展开模式，可以在游戏中实时输入命令并复制到剪贴板。

如何添加自定义命令模板？ ▼

通过拓展包功能添加自定义命令模板。创建 JSON 格式的拓展包文件，按照 Nexus Addon Spec 规范定义命令模板，然后在应用中导入即可。

ID 翻译库会更新吗？ ▼

是的，我们会定期更新 ID 翻译库以支持新版本的 Minecraft。你也可以通过拓展包自行添加或修改翻译内容。

Nexus 是开源的吗？ ▼

是的，Nexus 采用 MIT 许可证开源。你可以在 GitHub 上查看完整源码，也欢迎贡献代码或提交 Issue。
