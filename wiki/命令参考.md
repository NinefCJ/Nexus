# 命令参考

## 常用命令

### /give

```
/give <目标> <物品> [数量] [数据值] [组件]
```

给予玩家物品。

**示例**：
```
/give @p diamond_sword 1
/give @a iron_axe 1 0 {display:{Name:"锋利的斧头"}}
```

### /setblock

```
/setblock <x> <y> <z> <方块> [方块状态] [选项]
```

在指定位置放置方块。

**示例**：
```
/setblock ~ ~1 ~ minecraft:diamond_block
/setblock ~ ~ ~ minecraft:stone_button[face=wall,facing=south]
```

### /summon

```
/summon <实体> [x] [y] [z] [NBT]
```

召唤实体。

**示例**：
```
/summon zombie ~ ~1 ~
/summon minecraft:arrow ~ ~5 ~ {Motion:[0.0,-1.0,0.0]}
```

### /tp

```
/tp <目标> <目的地>
```

传送实体。

**示例**：
```
/tp @p ~ ~10 ~
/tp @e[type=zombie] @p
```

### /playsound

```
/playsound <音效> <目标> [x] [y] [z] [音量] [音高] [最小音量]
```

播放音效。

**示例**：
```
/playsound minecraft:entity.player.levelup @a
/playsound minecraft:block.note_block.piano @p ~ ~ ~ 1 1
```

### /particle

```
/particle <粒子名称> [位置] [偏移量] [速度] [数量] [模式] [玩家] [参数]
```

生成粒子效果。

**示例**：
```
/particle minecraft:flame ~ ~1 ~ 0.5 0.5 0.5 0.1 10
/particle minecraft:heart ~ ~ ~ 0 0 0 0.1 5 normal @p
```

### /playanimation

```
/playanimation <目标> <动画> [下一状态] [停止表达式] [控制器] [淡出时间]
```

播放动画。

**示例**：
```
/playanimation @s animation.player.wave
/playanimation @e[type=creeper] animation.creeper.swelling
```

### /hud

```
/hud <玩家> hide <HUD元素>
/hud <玩家> reset
```

控制 HUD 元素的显示。

**示例**：
```
/hud @p hide hotbar
/hud @a reset
```

## 目标选择器

| 选择器 | 描述 |
|--------|------|
| `@s` | 自己 |
| `@p` | 最近的玩家 |
| `@a` | 所有玩家 |
| `@e` | 所有实体 |
| `@r` | 随机玩家 |

### 选择器参数

```
@e[type=zombie, distance=0..10, name=Bob]
```

常用参数：

- `type`：实体类型
- `distance`：距离范围
- `name`：实体名称
- `tag`：实体标签
- `team`：队伍
- `score`：分数
- `x/y/z`：坐标
- `dx/dy/dz`：坐标范围

## 坐标系统

### 相对坐标

使用 `~` 表示相对于当前位置的坐标：

```
/tp ~ ~10 ~    # 向上传送10格
/setblock ~ ~-1 ~ minecraft:stone    # 在脚下放置石头
```

### 局部坐标

使用 `^` 表示相对于视线方向的坐标：

```
/tp ^ ^ ^5    # 向前传送5格
/summon arrow ^ ^ ^1    # 在前方1格召唤箭矢
```

### 绝对坐标

直接使用数字表示绝对坐标：

```
/tp 100 64 200    # 传送到坐标 (100, 64, 200)
```
