# 资源库

## 方块库

方块库包含 456 个基岩版方块，支持以下分类：

- 自然
- 装饰
- 功能
- 红色
- 建筑
- 运输
- 杂项

### 使用方法

1. 点击底部导航栏的"资源库"标签
2. 选择"方块"子标签
3. 使用搜索框搜索方块
4. 点击方块查看详情和命令生成

### 命令生成

选择方块后，系统会自动生成 `/setblock` 命令：

```
/setblock <x> <y> <z> <方块ID> [方块状态] [选项]
```

## 音效库

音效库包含 918 个基岩版音效，支持以下分类：

- 环境
- 生物
- 方块
- 玩家
- 物品
- 音乐

### 使用方法

1. 点击底部导航栏的"资源库"标签
2. 选择"音效"子标签
3. 使用搜索框搜索音效
4. 点击播放按钮试听音效

### 命令生成

选择音效后，系统会自动生成 `/playsound` 命令：

```
/playsound <音效ID> <目标> [x] [y] [z] [音量] [音高] [最小音量]
```
