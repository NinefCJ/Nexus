# 功能特性

## 命令补全

### 实时补全

输入命令时，系统会根据上下文提供智能补全建议。

### 语法高亮

不同类型的 token 使用不同颜色显示：

- **命令**：紫色
- **选择器**：蓝色
- **坐标**：绿色
- **字符串**：橙色
- **数字**：红色

### 错误检测

实时检测命令语法错误，并提供错误原因和修复建议。

## 旧命令转换

### 1.18 → 1.20+ 转换

自动将旧命令转换为新语法：

- 方块数据值 → 方块状态
- execute 旧语法 → 新语法
- 旧格式 NBT → 新格式 NBT

### 输入法模式

开启输入法模式后，输入时自动转换命令，无需手动操作。

## 命令穷举

批量生成参数组合命令，方便测试和调试。

## 命令库

### 本地命令库

创建和管理本地命令集合，支持分类和搜索。

### 云端命令库

浏览和下载社区分享的命令，也可以分享自己的命令。

## 悬浮窗

在游戏中使用命令助手，一键复制命令到剪贴板。

## 游龙流编辑器

可视化编程，用流程图方式创建命令链，支持条件分支和循环。

## 主题系统

多种预设主题可选，支持自定义背景图片和毛玻璃效果。
