# 命令参考

## 常用命令

### /give

```
/give <目标> <物品> [数量] [数据值] [组件]
```

给予玩家物品。

### /setblock

```
/setblock <x> <y> <z> <方块> [方块状态] [选项]
```

在指定位置放置方块。

### /summon

```
/summon <实体> [x] [y] [z] [NBT]
```

召唤实体。

### /tp

```
/tp <目标> <目的地>
```

传送实体。

### /playsound

```
/playsound <音效> <目标> [x] [y] [z] [音量] [音高] [最小音量]
```

播放音效。

### /particle

```
/particle <粒子名称> [位置] [偏移量] [速度] [数量] [模式] [玩家] [参数]
```

生成粒子效果。

### /playanimation

```
/playanimation <目标> <动画> [下一状态] [停止表达式] [控制器] [淡出时间]
```

播放动画。

### /hud

```
/hud <玩家> hide <HUD元素>
/hud <玩家> reset
```

控制 HUD 元素的显示。

## 目标选择器

| 选择器 | 描述 |
|--------|------|
| `@s` | 自己 |
| `@p` | 最近的玩家 |
| `@a` | 所有玩家 |
| `@e` | 所有实体 |
| `@r` | 随机玩家 |

### 选择器参数

```
@e[type=zombie, distance=0..10]
```

常用参数：

- `type`：实体类型
- `distance`：距离范围
- `name`：实体名称
- `tag`：实体标签
- `team`：队伍
- `score`：分数
- `x/y/z`：坐标
- `dx/dy/dz`：坐标范围
